# AI_Dashborad_Streamlit
AI 데이터 시각화 대시보드


## 가상 환경 만들기 (선택)
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

## 필요한 패키지 설치
pip install streamlit pandas matplotlib

## requirements.txt 설치
pip freeze > requirements.txt

## https://streamlit.io/cloud 접속
new app 생성 및 deploy

## dev 브랜치 만들기

### main 브랜치 최신 상태에서 dev 브랜치 생성
git checkout -b dev

### GitHub 원격에 push
git push origin dev
